# Caf-expresso
O Café Expresso é um sistema de autoatendimento para uma cafeteria de pequeno porte. O objetivo é permitir que os clientes realizem seus pedidos em um totem digital, sem precisar enfrentar filas, tornando o atendimento mais rápido e eficiente.  O sistema também auxilia os atendentes, permitindo visualizar e gerenciar os pedidos em tempo real.
